# LRN Wi-Fi電波測定ツール

介護施設の各居室でWi-Fi電波強度を測定するWindows向けツールです。
営業・カスタマーサクセスなど、技術に詳しくないスタッフでも簡単に使えます。

## 使い方

1. **WiFi_Survey.bat** をダブルクリック
2. 施設名を入力 → 周辺Wi-Fiをスキャンし、番号付きでSSID一覧を表示
3. 測定するSSIDの番号を選択（Enterで接続中SSIDを自動選択）
4. **居室に入り、扉を閉めた状態**で居室名を入力 → 自動で10回スキャン計測（約20秒）
5. 結果表示後、次の居室名を入力（終了は `q`）
6. 終了すると以下が自動生成されます：
   - **CSV** (Shift-JIS/Excel対応) — スプレッドシートで分析用
   - **HTMLレポート** — 居室別測定結果・周辺ネットワーク一覧・チャネル利用状況（NW担当者向け参考情報）

## 必要環境

- Windows 10 以降
- Wi-Fi接続中のノートPC
- PowerShell 5.1 以降（Windows標準搭載）

## 測定項目

| 項目 | 説明 |
|------|------|
| 電波強度 | RSSI（dBm）（10回スキャンの平均・最小・最大） |
| SSID | 測定対象のWi-Fiネットワーク名 |
| チャンネル | 使用チャンネル番号 |
| 周波数帯 | 2.4GHz / 5GHz |
| 無線規格 | 802.11a/b/g/n/ac/ax |
| 周辺ネットワーク | 各居室でのスキャン結果（AP一覧） |
| チャネル利用状況 | 各チャネルのAP台数（NW担当者への参考情報） |

## 電波強度の目安（社内目安値：-70 dBm）

測定はスリープセンサゲートウェイを設置するベッドフレーム付近で、扉を閉めた状態で行ってください。

| 評価 | RSSI（dBm） | 社内基準 |
|------|------------|---------|
| 優良 | -65 dBm 以上 | 社内目安値を大きく上回る |
| 良好 | -70 dBm 以上 | 社内目安値を満たす |
| 普通 | -80 dBm 以上 | 社内目安値（-70 dBm）未達 → NW担当者に確認 |
| 弱い | -90 dBm 以上 | 社内目安値を大きく下回る → NW担当者に確認 |
| 非常に弱い | -90 dBm 未満 | 電波がほとんど届かない → NW担当者に確認 |

> ※ チャネル干渉の評価は業界共通の閾値が存在しないため、当社では実施しません。
> チャネル利用状況はNW担当者への参考情報として表示しています。
