# LRN Wi-Fi電波測定ツール

介護施設の各居室でWi-Fi電波強度を測定するWindows向けツールです。  
営業・CSやお客様など、ターミナル操作に慣れていない方でも簡単に使えます。

## 使い方

1. **WiFi_Survey.bat** をダブルクリック
2. 施設名を入力
3. 居室名を入力 → 自動で10回測定（約20秒）
4. 結果表示後、次の居室名を入力（終了は `q`）
5. 終了すると以下が自動生成されます：
   - **CSV** (Shift-JIS/Excel対応) — スプレッドシートで分析用
   - **HTMLレポート** — ブラウザで表示、PDFとして印刷可能

## 必要環境

- Windows 10 以降
- Wi-Fi接続中のノートPC
- PowerShell 5.1 以降（Windows標準搭載）

## 測定項目

| 項目 | 説明 |
|------|------|
| 電波強度 | 0〜100%（10回測定の平均・最小・最大） |
| SSID | 接続中のWi-Fiネットワーク名 |
| チャンネル | 使用チャンネル番号 |
| 周波数帯 | 2.4GHz / 5GHz |
| 無線規格 | 802.11a/b/g/n/ac/ax |

## 電波強度の目安

| 評価 | 強度 | 説明 |
|------|------|------|
| 優良 | 80%以上 | 非常に安定した通信が可能 |
| 良好 | 60〜79% | 安定した通信が可能 |
| 普通 | 40〜59% | 基本的な通信は可能 |
| 弱い | 20〜39% | 通信が不安定になる可能性 |
| 非常に弱い | 20%未満 | AP増設を推奨 |

## ファイル構成

```
WiFi_Survey.bat   ← ダブルクリックで起動（エントリーポイント）
WiFi_Survey.ps1   ← メインスクリプト
```
